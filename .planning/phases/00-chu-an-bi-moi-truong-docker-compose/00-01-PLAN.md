<task type="auto">
  <name>Task 2: Tạo file docker-compose.yml cấu hình các service hạ tầng</name>
  <files>docker-compose.yml</files>
  <read_first>init.sql</read_first>
  <action>
    Tạo file `docker-compose.yml` tại thư mục gốc dự án để quản lý 4 container sau:
    1. **postgres**: 
       - Image: `postgres:15-alpine`
       - Ports: `5432:5432`
       - Environment: `POSTGRES_USER=postgres`, `POSTGRES_PASSWORD=postgres`
       - Volumes: Mount file `./init.sql` vào `/docker-entrypoint-initdb.d/init.sql` để chạy lúc khởi tạo.
    2. **redis**:
       - Image: `redis:7-alpine`
       - Ports: `6379:6379`
    3. **zookeeper**:
       - Image: `confluentinc/cp-zookeeper:7.4.0`
       - Environment: `ZOOKEEPER_CLIENT_PORT=2181`
    4. **kafka**:
       - Image: `confluentinc/cp-kafka:7.4.0`
       - Ports: `9092:9092`
       - Environment:
         - `KAFKA_ZOOKEEPER_CONNECT=zookeeper:2181`
         - `KAFKA_ADVERTISED_LISTENERS=PLAINTEXT://localhost:9092,PLAINTEXT_INTERNAL://kafka:29092`
         - `KAFKA_LISTENER_SECURITY_PROTOCOL_MAP=PLAINTEXT:PLAINTEXT,PLAINTEXT_INTERNAL:PLAINTEXT`
         - `KAFKA_INTER_BROKER_LISTENER_NAME=PLAINTEXT_INTERNAL`
         - `KAFKA_OFFSETS_TOPIC_REPLICATION_FACTOR=1`
  </action>
  <verify>Kiểm tra cú pháp file docker-compose.yml thông qua lệnh `docker compose config` hoặc đọc nội dung.</verify>
  <acceptance_criteria>
    - docker-compose.yml chứa cấu hình postgres port 5432:5432
    - docker-compose.yml chứa cấu hình redis port 6379:6379
    - docker-compose.yml chứa cấu hình kafka port 9092:9092 và zookeeper
    - Volume mapping cho init.sql được định nghĩa chính xác
  </acceptance_criteria>
</task>
